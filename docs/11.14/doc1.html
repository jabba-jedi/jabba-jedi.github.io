Kraken evaluates rules in order. Order is calculated from default rules depending on each other. Other rules are evaluated after the default rules. Rule dependencies are calculated from expressions. In a default rule, dependency is extracted from Condition Expression and Default Value Expression. In this example rule will depend on default rules, that are defined on Policy.policyNumber and Policy.defaultValue.
